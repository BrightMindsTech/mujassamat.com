# Mujassamat - 3D Printing Services

A modern, dark-themed e-commerce website for Mujassamat 3D printing services in Jordan. Built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- **Multi-step Quote Form**: Comprehensive form with file upload, 3D preview, and detailed specifications
- **File Upload**: Support for STL and OBJ files with client-side validation
- **3D Preview**: Live preview of uploaded STL files using Three.js
- **Gallery**: Filterable project gallery with modal previews
- **Email Integration**: Multiple email options (mailto, EmailJS, Nodemailer)
- **Admin Panel**: Submission management with status tracking
- **Dark Theme**: Modern dark design with blue/purple/cyan gradients
- **Responsive**: Mobile-first responsive design
- **Accessibility**: ARIA labels, keyboard navigation, and screen reader support

## Tech Stack

### Frontend
- **React 18** with TypeScript
- **Wouter** for routing
- **Tailwind CSS** for styling
- **Shadcn UI** components
- **Framer Motion** for animations
- **Three.js** for 3D file preview
- **React Hook Form** with Zod validation
- **TanStack Query** for data fetching

### Backend
- **Express.js** with TypeScript
- **Multer** for file uploads
- **Zod** for validation
- **File system** for data storage
- **Rate limiting** for API protection
